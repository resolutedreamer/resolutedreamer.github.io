Home Dashboard

- Configured a home for real-time monitoring of water and power by aggregating multiple sensors streams with Python
- Created a web dashboard using Django and Bootstrap Frameworks to:
- Visualize historical trends using data stored in sMAP and SQL databases running on Linux Virtual Machines
- Configure email and phone notifications based on consumption limits
- Enabled web access to Philips Hue home lighting system
- Used Git version control (BitBucket) and Skype for team collaboration

SmartPakku

- Open Source backpack, phone app, and website that monitors a child’s backpack use for optimal back health
- Designed low cost, easy to assemble hardware so that it can be fitted into any standard backpack with hip belt
- Information is streamed over Bluetooth LE to a Windows Phone app which will suggest fitting adjustments
- App connects to a MongoLab Database hosted on Microsoft Azure’s cloud service to update the stored data
- Data is then accessible by parent at a website (created using PHP)

Bluetooth LE UART 8001

- First Windows Phone app that supports the Nordic Semiconductors custom UART service
- Developed as a result of insufficient existing Bluetooth resources for Windows 8.1

Embedded Synchronization

- A prototypical IoT system following the Device-Gateway-Cloud model
- Two mbed Freedom Development Platforms (FRDM-KL46Z) used as the Device
- Raspberry Pi used as a gateway to synchronize the two mbeds with NTP time
- Cloud based backend using MongoDB to store the sensing for web display and actuation

NeuroPass

A secure neural password based on EEG

- Use raw data from Emotiv EEG Neuroheadset to identify facial movement and authenticate users
- Developed digital filters and algorithms to remove imperfections and noise to identify key signal features
- Signal processing done in C on a TI C6416 DSP, facing constrained memory and processor constraints
- Presented as final project for Senior Digital Signal Processing Design Course
